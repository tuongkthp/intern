# Tạo secret cho nova
Gen uuid

    uuidgen

Tạo file secret.xml

    <secret ephemeral='no' private='no'>
        <uuid>[UUID]</uuid>
        <usage type='ceph'>
            <name>[name] secret</name>
        </usage>
    </secret>

Define secret

    virsh secret-define --file secret.xml

Lấy key của user truy cập ceph:

    ceph auth get-key client.cinder

Thêm key cho UUID:

    virsh secret-set-value --secret [UUID] --base64 [key]
