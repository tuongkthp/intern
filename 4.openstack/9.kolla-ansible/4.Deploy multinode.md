# 1. Chuẩn bị
### 1.1 Chuẩn bị server (vm)
VM1 (Controller):
- OS: ubuntu-24-04

- Ram: 20GB

- Disk: 100GB

- 2 interface:
    + ens3: 10.2.2.209 

    + ens7: 192.168.20.63

VM2 (Compute):
- OS: ubuntu-24-04

- Ram: 20GB

- Disk: 100GB

- 2 interface:
    + ens3: 10.2.2.219 
    
    + ens7: 192.168.20.252

### 1.2 Cấu hình truy cập
Set hostname:

    # VM1
    hostnamectl set-hostname control 

    # VM2
    hostnamectl set-hostname compute1

Add host trên cả 2 vm (/etc/hosts):

    10.2.2.209 control
    10.2.2.219 compute1

Thiết lập để có thể truy cập từ xa giữa 2 vm bằng ssh (user root):

    # Tạo key
    ssh-keygen

    # Add public key của vm1 vào vm2 và ngược lại vào file /root/.ssh/authorized_keys

Thử truy cập giữa 2 vm với user root mà ko cần nhập mật khẩu:

    # VM1
    ssh compute1

    # VM2
    ssh control1

### 1.3 Cài gói cần thiết
Cài docker trên cả 2 vm:

    apt update
    apt install docker.io -y
    systemctl enable docker
    systemctl start docker

#### Trên VM1
Cài Python build dependencies:

    sudo apt install git python3-dev libffi-dev gcc libssl-dev libdbus-glib-1-dev

Cài venv:

    apt install python3-venv

Tạo môi trường ảo:

    python3 -m venv /path/to/venv
    source /path/to/venv/bin/activate

Đảm bảo đã cài phiên bản mới nhất của pip:

    pip install -U pip

Cài Kolla-ansible:

    git clone --branch stable/2025.2 https://opendev.org/openstack/kolla-ansible

    pip install -e ./kolla-ansible
    sudo mkdir -p /etc/kolla
    sudo chown $USER:$USER /etc/kolla
    cp -r kolla-ansible/etc/kolla/* /etc/kolla
    cp kolla-ansible/ansible/inventory/* .

Cài các yêu cầu Ansible Galaxy:

    kolla-ansible install-deps

Kolla password:

    cd kolla-ansible/tools
    ./generate_passwords.py

### 1.4 Cấu hình Kolla (VM1)
/etc/global.yml:

    kolla_base_distro: "ubuntu"
    kolla_internal_vip_address: "10.2.2.220"
    network_interface: "ens3"
    neutron_external_interface: "ens7"

Nếu sử dụng haproxy với internal_vip_address là 10.2.2.220, vm enable port security cần allowed_address_pairs cho ip 10.2.2.220:

    openstack port set 473f0b04-398e-4b80-87a7-7806860b9000 --allowed-address ip-address=10.2.2.220

File multinode:

    [control]
    control

    [network]
    control
    compute1

    [compute]
    compute1

    [monitoring]
    control

    [storage]
    control

    [deployment]
    control

# 2 Deploy
Bootstrap servers (VM1):

    kolla-ansible bootstrap-servers -i multinode

Prechecks:

    kolla-ansible prechecks -i multinode

Deployment:
    
    kolla-ansible deploy -i multinode

### Openstack
Cài OpenStack CLI:

    pip install python-openstackclient -c https://releases.openstack.org/constraints/upper/2025.2

Genarate file clouds.yaml để tạo thông tin user Openstack:

    kolla-ansible post-deploy
