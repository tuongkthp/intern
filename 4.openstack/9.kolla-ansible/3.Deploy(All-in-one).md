# 1. Chuẩn bị 
Host hoặc VM yêu cầu tối thiểu:
- 2 network interface
    + ens3: 10.2.4.208

    + ens7: 192.168.20.161

- 8 GB ram

- 40 GB disk

- OS: ubuntu-24-04 trở lên

Note: Với host và vm điều khiển từ xa cần tối thiểu 3 interface
- Interface 1: Interface để add vào bridge ovs

- Interface 2: Có ip dùng cho openstack service giao tiếp internal

- Interface 3: Dùng để truy cập từ xa (Tránh bị mất kết nối)

# 2. Triển khai
### 2.1 Cài dependencies
Update hệ thống:
    
    sudo apt update

Cài các dependencies:

    sudo apt install git python3-dev libffi-dev gcc libssl-dev libdbus-glib-1-dev

### 2.2 Cài dependencies cho môi trường ảo
Cài dependencies môi trường ảo:

    sudo apt install python3-venv

Tạo một môi trường ảo và kích hoạt nó:

    python3 -m venv /path/to/venv
    source /path/to/venv/bin/activate

Đảm bảo đã cài đặt phiên bản mới nhất của pip:

    pip install -U pip

### 2.3 Install Kolla-ansible
Clone kolla-ansible từ git:

    git clone --branch stable/2025.2 https://opendev.org/openstack/kolla-ansible

Cài đặt những yêu cầu của kolla và kolla-ansible:

    pip install -e ./kolla-ansible

Tạo thư mục /etc/kolla

    sudo mkdir -p /etc/kolla
    sudo chown $USER:$USER /etc/kolla

Copy file cấu hình đến thư mục /etc/kolla:

    cp -r kolla-ansible/etc/kolla/* /etc/kolla

Copy inventory file đến thư mục hiện tại (all-in-one and multinode):

    cp kolla-ansible/ansible/inventory/* .

Install Ansible Galaxy dependencies:

    kolla-ansible install-deps

### 2.4 Chuẩn bị cấu hình ban đầu
Kolla passwords:

    cd kolla-ansible/tools
    ./generate_passwords.py

Kolla globals.yml (/etc/kolla/globals.yml):

    kolla_base_distro: "ubuntu"
    network_interface: "ens7"
    neutron_external_interface: "ens3"
    # vip address: ip internal giúp tạo endpoint cho openstack services
    kolla_internal_vip_address: "192.168.20.250"  

### 2.5 Deploy
Bootstrap servers with kolla deploy dependencies:

    kolla-ansible bootstrap-servers -i ../all-in-one

Do pre-deployment checks for hosts:

    kolla-ansible prechecks -i ../all-in-one

Finally proceed to actual OpenStack deployment:

    kolla-ansible deploy -i ../all-in-one

### 2.6 Openstack CLI client
Install the OpenStack CLI client:

    pip install python-openstackclient -c https://releases.openstack.org/constraints/upper/2025.2

Tạo file cho user admin:

    kolla-ansible post-deploy
