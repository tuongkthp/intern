# Network interface metrics
- Rx (Receive): Đo số gói nhận được bởi giao diện mạng (bytes)

- Tx (Transmit): Đo số gói chuyển đi bởi giao diện mạng (bytes)

- Throughput: Thể hiện tốc độ truyền dữ liệu thực tế qua một giao diện mạng, bằng tổng số gói đã nhận và truyền chia cho khoảng thời gian

- Bandwidth: Tốc độ truyền dữ liệu tối đa của một kết nối mạng (bps, Mbps, Gbps)

- Các câu lệnh/công cụ kiểm tra thông số mạng:
    * netstat
    
    * nuttcp

    * iperf3
