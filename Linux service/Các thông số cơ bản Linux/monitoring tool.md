| Công cụ | Chức năng |
| ----------- | -------- |
| top | Hiển thị thông tin các tiến trình đang chạy trên hệ thống (thời gian thực) |
| uptime | Theo dõi và quản lý hiệu suất của hệ thống (thời gian hoạt động, số lượng người dùng, load trung bình,...) |
| vmstat | Theo dõi và phân tích hiệu suất hệ thống một cách chi tiết |
| ps | Xem và quản lý các tiến trình đang chạy trên hệ thống |
| pstree | Hiển thị về cấu trúc process và cách chúng tương tác với nhau |
| free | Kiểm tra tình trạng sử dụng bộ nhớ của hệ thống |
| netstat | Hiển thị thông tin chi tiết về các kết nối mạng, cổng đang lắng nghe, bảng định tuyến và nhiều thông tin liên quan đến mạng khác |
| tcpdump | Bắt và phân tích lưu lượng mạng đi qua một giao diện mạng |
