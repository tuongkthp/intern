# Database
- Cơ sở dữ liệu là một tập hợp dữ liệu có tổ chức được lưu trữ dưới dạng điện tử để tạo điều kiện dễ dàng truy cập, quản lý và thao tác. Nó có thể lưu trữ nhiều loại dữ liệu khác nhau, chẳng hạn như văn bản, số, hình ảnh, video

- Tính năng:
    * Lưu trữ dữ liệu hiệu quả

    * Thao tác với dữ liệu

    * Tính toàn vẹn và bảo mật dữ liệu

    * Khả năng mở rộng

# Các loại database
- Cơ sở dữ liệu quan hệ:
    * Dữ liệu được tổ chức dưới dạng bảng với hàng và cột

    * Sử dụng lược đồ để định nghĩa cách dữ liệu được cấu trúc và cách các bảng khác nhau liên quan tới nhau

    * Cấu trước dựa trên lược đồ nghiêm ngặt

    * Khóa chính và khóa ngoại

    * Phù hợp cho dữ liệu có cấu trúc

    * Một số cơ sở dữ liệu quan hệ:
        + MySQL

        + PostgreSQL

        + Microsoft SQL Server

- Cơ sở dữ liệu phi quan hệ:
    * Được thiết kế đê xử lý dữ liệu phi cấu trúc và bán cấu trúc
